== Description ==
 
Displays save for later icon on posts, pages or custom post types, you can save and remove any post from the saved collection. Reader can view the saved for later reading collectin by clicking the icon on the bottom of the browser.User will be redirected to the page containing all the saved Articles or posts and can read and delete the item form the collection.

<h3>Features:</h3>
1. Read me later icon for post, page and custom post types.
2. you can select the icon/image for Read Me Later Icon.
3. Shortcode support to add Read Me Later icon any where in content of specific post or page.
4. Show Read Me Later icon for loggedin users or  visitors.
5. Translation ready.
1. **More to come...**


<h3>HOW TO USE THE PLUGIN</h3>
 Just install and Go to Read Me Later Settings page select position of icon, select icon from icon list and you will see icon on pages / posts. You can also display Read Me Later icon by using simple shortcode [Readmelater]

<h3>WANT TO CUSTOMIZE?</h3>
You can change the icon from dashboard -> Read Me Later -> Display and select any Icon from list.

<h3>Installation</h3>
1. Install the plugin from wordpress directory OR
   Manually Upload `read-me-later folder` to the `/wp-content/plugins/` directory and upzip it
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to plugin settings page and check/select form the options where and how to want to display the read me later icon Check post detail page to see it in action.
